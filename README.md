## GETTING STARTED

1.  Clone this repo and open in a code editor

        git clone https://github.com/maxtkarlsson/Webgame-SQL-API.git

2.  install

    npm i

3.  run in terminal

    npm run seedDb

    npm run dev

4.  Create a .env-file and add

    PORT = 3000

    JWT_SECRET = (your jwt secret)
